# Monotile

[![CI](https://github.com/ctkrug/monotile/actions/workflows/ci.yml/badge.svg)](https://github.com/ctkrug/monotile/actions/workflows/ci.yml)

An interactive studio for the 2023 **einstein** discovery — the "hat" and "spectre" tiles,
the first shapes ever found that tile the infinite plane using only one shape, with no
mirror image, and *never* repeating.

Pan around a canvas of tiles generated by the substitution system that produces the hat/spectre
monotiling, watch new tiles materialize forever without the pattern ever looping back on itself,
then recolor the whole thing into a poster and export it as clean SVG.

## Why

The aperiodic monotile problem — does a single shape exist that tiles the plane but never
periodically — was open for over 50 years until David Smith, Joseph Samuel Myers, Craig S.
Kaplan, and Chaim Goodman-Strauss found the "hat" (and its chiral-free cousin, the "spectre")
in 2023. It's a genuinely new piece of mathematics, and almost nobody outside of a few papers
and blog posts has ever *seen* it move. Monotile is a place to play with it directly: drag to
explore an infinite non-repeating tiling, and take a piece of it home as art.

## The wow moment

Drag the canvas and a single curvy tile keeps filling space forever — visibly never
repeating — then recolor it into a poster in one click.

## Features

- **Substitution-based infinite generation** — tiles are produced by recursively inflating and
  subdividing metatiles per the spectre substitution system, so the pattern can grow without
  bound in any direction as you pan.
- **Smooth infinite pan and cursor-anchored zoom** — drag/scroll to explore; only the tiles
  inside (and just outside) the viewport are computed and drawn.
- **Coloring schemes** — recolor every tile by orientation, substitution generation, or supertile
  membership (or keep the default monochrome line art), with a cross-fade ripple animation that
  radiates outward from wherever you click.
- **Poster-quality SVG export** — export the current view as a clean, standalone vector file with
  no external references, ready to print.
- **Synth sound + mute** — a soft chime on recolor and a shutter click on export, generated
  entirely in WebAudio (no audio files); mute your preference and it persists across reloads.
- **Tile inspector** — a live crosshair and coordinate/generation/type readout follow your
  cursor; click a tile to pin an inspector panel with its type, generation, orientation, and
  supertile lineage.
- **Touch and mobile** — single-finger drag pans and a two-finger pinch zooms on touch devices;
  below 390px the control panel collapses into a bottom sheet so the tiling keeps the screen.
- **Accessible by default** — every control has themed hover/focus/active states and a ≥44px
  touch target, the export toast is an ARIA live region, and keyboard tab order is logical.

## Stack

Vanilla JavaScript + HTML5 Canvas, bundled and served with [Vite](https://vitejs.dev/). No UI
framework — the whole surface area is a single canvas and a slim control panel, so a framework
would only add weight. [Vitest](https://vitest.dev/) covers the geometry and substitution logic.
The build is a static site (no server, no backend) so it can be hosted from any static host,
including a subpath.

## Status

The core loop is live end-to-end: drag or pinch to pan/zoom an infinite, genuinely non-repeating
spectre tiling on desktop or touch, recolor it by orientation/generation/supertile with an
animated ripple, export the current view as a poster-ready SVG, and inspect any tile's type/
generation/lineage. A final design ship-gate pass is still ahead — see
[`docs/VISION.md`](docs/VISION.md) for the full design,
[`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) for how it's built, and
[`docs/BACKLOG.md`](docs/BACKLOG.md) for the build plan.
